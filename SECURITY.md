# Security Policy

This is an educational practice repository — it ships no runtime service, network code, or
production dependencies beyond test-time libraries. Still, if you spot anything that could harm a
user who clones and runs it (e.g. an unsafe dependency, a malicious test fixture), we want to know.

## Supported Versions

The `main` branch is the only supported version. Fixes land there.

## Reporting a Vulnerability

Please **do not** open a public issue for security reports. Instead, use one of these private
channels:

- Open a [GitHub private security advisory](https://github.com/svrohith9/java-streams-playground/security/advisories/new)
- Or email **svrohith9@gmail.com** with the details and steps to reproduce.

We aim to acknowledge reports within a few days and will credit you in the fix unless you prefer to
remain anonymous.
